Aplikasi BMI Calculator adalah platform kesehatan mobile yang dirancang untuk membantu pengguna memantau indeks massa tubuh (BMI) secara mandiri dan praktis. Aplikasi ini menyediakan alur kerja yang intuitif mulai dari pembukaan aplikasi hingga analisis kesehatan berbasis AI.

## 🚀 Fitur Utama
* **Splash Screen Interaktif**: Memberikan pengalaman pembukaan aplikasi yang estetik dengan transisi logo.
* <img width="393" height="852" alt="image" src="https://github.com/user-attachments/assets/df66313a-6c30-44e9-ac28-ba6c686b38cd" />

* **Kalkulator BMI**: Perhitungan BMI yang akurat berdasarkan input tinggi dan berat badan pengguna.
* **Riwayat Kesehatan**: Fitur penyimpanan hasil perhitungan untuk memantau perkembangan berat badan dari waktu ke waktu.
* **AI Assistant**: Konsultasi cerdas mengenai informasi BMI dan tips kesehatan secara *real-time*.
* **Fitur Scan**: Dukungan pemindaian berat badan digital untuk kemudahan input data.

## 🛠 Teknologi & Alat Pengembangan
* **Desain & Prototyping**: 
    * **Wireframe**: Digunakan untuk merancang kerangka dasar alur aplikasi.
    * **Storyboard**: Digunakan untuk memetakan interaksi pengguna langkah-demi-langkah.
* **Manajemen Tugas**: ClickUp (digunakan untuk melacak progress pengembangan aplikasi).

## 💡 Penjelasan Alur Aplikasi
1. **Inisialisasi**: Aplikasi dimulai dengan rangkaian *splash screen* yang memuat logo.
2. **Input Data**: Pengguna memasukkan tinggi dan berat badan untuk dihitung nilai BMI-nya.
3. **Hasil & Riwayat**: Sistem menampilkan status BMI (kurus/normal/overweight) yang dapat disimpan ke dalam riwayat pengguna.
4. **Bantuan AI**: Pengguna dapat berinteraksi dengan AI Assistant untuk mendapatkan jawaban seputar kesehatan.
5. **Pengelolaan Akun**: Pengguna dapat mengakses menu akun untuk mengatur profil dan preferensi aplikasi.
