# 📖 Deskripsi

**BuleFit BMI Calculator** merupakan aplikasi mobile yang dirancang untuk membantu pengguna menghitung **Body Mass Index (BMI)** secara cepat, mudah, dan akurat. Selain menghitung BMI, aplikasi ini juga menyediakan riwayat hasil perhitungan serta AI Assistant yang dapat memberikan informasi mengenai kategori BMI dan tips kesehatan.

Aplikasi memiliki tampilan modern dengan alur penggunaan yang sederhana sehingga memudahkan pengguna dalam memantau kondisi berat badan mereka.

## ⚖️ BMI Calculator

Fitur utama aplikasi yang digunakan untuk menghitung nilai Body Mass Index berdasarkan:

- Berat badan (kg)
- Tinggi badan (cm)

Setelah proses perhitungan selesai, aplikasi akan menampilkan:

- Nilai BMI
- Kategori BMI
- Status kesehatan pengguna

# 📱 Alur Aplikasi

1. Splash Screen muncul saat aplikasi dijalankan.
2. Pengguna masuk ke halaman utama.
3. Pengguna mengisi:
   - Tinggi badan
   - Berat badan
4. Sistem menghitung nilai BMI.
5. Hasil BMI beserta kategorinya ditampilkan.
6. Pengguna dapat menyimpan hasil ke menu **History**.
7. Pengguna dapat bertanya kepada **AI Assistant** mengenai hasil BMI maupun tips kesehatan.
8. Pengguna dapat mengakses halaman **Profile** untuk melihat informasi akun.

# 📊 Kesimpulan

**BuleFit BMI Calculator** merupakan aplikasi mobile yang menggabungkan perhitungan **Body Mass Index (BMI)** dengan fitur **AI Assistant** untuk memberikan pengalaman yang lebih interaktif kepada pengguna. Melalui fitur perhitungan BMI, riwayat hasil, serta konsultasi AI, aplikasi ini dapat membantu pengguna memantau kondisi tubuh dan memperoleh informasi kesehatan secara praktis.

Dengan desain antarmuka yang sederhana, modern, dan mudah digunakan, aplikasi ini diharapkan mampu menjadi media pendukung dalam meningkatkan kesadaran masyarakat terhadap pentingnya menjaga berat badan ideal dan menerapkan gaya hidup sehat.
